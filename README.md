# sjtmp-vault-template

A template for a personalized Zettelkasten workflow:  
**Seed → Journal → Theme → Meta → Publish**

This repository provides a minimal folder structure and note templates
to scaffold your own SJTMP knowledge workflow using Obsidian (or similar tools).


## Directory Structure

```

00_Seed/      — raw notes / inbox / quick capture
01_Journal/   — daily logs / observations
02_Theme/     — recurrent topics
├ Private/ — personal themes
└ Public/  — generalizable themes
03_Meta/      — structured insights
04_Publish/   — external outputs
templates/    — note templates

```

> This structure mirrors the intended flow of knowledge:
> **Seed → Journal → Theme → Meta → Publish**


## Usage

1. Click **“Use this template”** to create your private repository  
2. Clone it locally  
3. Open the folder as a Vault in Obsidian  
4. Start jotting raw ideas in `00_Seed/`  
5. Promote them through the layers as they evolve
